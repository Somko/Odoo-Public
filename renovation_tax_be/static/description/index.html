Change the VAT tax with one mouse click from 21% to 6%

This module has been developed for the Belgian building sector for which a reduced VAT percentage may be used in specific renovation cases. With this module you can easily modifiy the VAT percentage with one mouse click of all invoicelines from 21 % to 6% and vice versa.

From the invoices overview, you can instantly see alle invoice that are based on the reduced VAT percentage.

Print VAT certificates

With this module, it is also possible to print the VAT certificate for tyhe customers which they have to sign to declare that are entitled to the reduced VAT percentage. This PDF certificate will contain the invoice number and the invoice date so that the administration is more efficient.
